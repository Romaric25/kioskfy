# MaxMind GeoLite2 Database

Ce dossier contient la base de données GeoLite2 pour la géolocalisation IP.

## Configuration

1. Créez un compte gratuit sur [MaxMind](https://www.maxmind.com/en/geolite2/signup)
2. Générez une clé de licence dans votre compte
3. Téléchargez la base de données **GeoLite2-Country** (format .mmdb)
4. Placez le fichier `GeoLite2-Country.mmdb` dans ce dossier

## Note importante

Le fichier `GeoLite2-Country.mmdb` n'est PAS inclus dans le dépôt Git pour des raisons de licence.
Vous devez le télécharger manuellement.

## Mise à jour

MaxMind met à jour sa base de données chaque semaine. Il est recommandé de la mettre à jour régulièrement
pour une meilleure précision.
